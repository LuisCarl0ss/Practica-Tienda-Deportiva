Tienda Deportiva hecha con WordPress

Tema elegido: Venta de articulos deportivos

Esta es una tienda online que ofrece productos deportivos, para corredores, amantes del gimasio,
amantes del yoga, etc. En esta tienda, encontraras todo lo que necesitas para que tus sesiones de
entrenamiento, sean lo más eficientes posibles.

Se usó LocalWP para ejecutar la web y es necesario importar el archivo .sql cada que haya cambios
